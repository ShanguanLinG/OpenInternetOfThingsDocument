# 各个设备的IP设置

## 一. 路由器配置

### 路由器基本配置

1. **重置路由器**并将网线连接到路由器
2. **访问路由器管理界面**: 浏览器访问 `192.168.1.1`, 密码为 `123456`
3. **配置上网设置**: 点击路由设置->上网设置->基本设置

| 配置项      | 设置值             |
|----------|-----------------|
| 连接类型     | 固定IP地址          |
| IP地址     | 192.168.0.<工位号> |
| 子网掩码     | 255.255.255.0   |
| 网关       | 192.168.0.254   |
| 首选DNS服务器 | 同网关             |
| 备用DNS服务器 | 不填              |

4. **点击保存**
5. **配置LAN口设置**: 点击LAN口设置

| 配置项      | 设置值            |
|----------|----------------|
| LAN口IP设置 | 手动             |
| IP地址     | 172.18.<工位号>.1 |
| 子网掩码     | 255.255.255.0  |

6. **配置完毕**

## 二. 服务器IP地址配置

### Windows服务器配置

1. **打开服务器**(一台电脑)
2. **进入网络连接**: 控制面板\所有控制面板项\网络连接
3. **配置TCP/IPv4**: 右键属性, 双击Internet协议版本4(TCP/IPv4)
4. **选择使用下面的IP地址**:

| 配置项      | 设置值                              |
|----------|----------------------------------|
| IP地址     | 172.18.<工位号>.11                  |
| 子网掩码     | 255.255.255.0(必须和路由器LAN口子网掩码一致)  |
| 默认网关     | 172.18.<工位号>.1(必须和路由器LAN口IP地址一致) |
| 首选DNS服务器 | 同默认网关                            |
| 备用DNS服务器 | 不填                               |

5. **配置完毕**

| 默认网关     | 与服务器配置相同        |
| 首选DNS服务器 | 与服务器配置相同        |

## 四. 串口服务器配置

### 串口服务器网络配置

1. **重置串口服务器**
2. **将电脑网络调制与串口服务器同一网段**:
    * 进入控制面板\所有控制面板项\网络连接
    * 右键属性, 双击Internet协议版本4(TCP/IPv4)
    * 选择使用下面的IP地址:

| 配置项      | 设置值                                                       |
|----------|-----------------------------------------------------------|
| IP地址     | 192.168.14.<任意值>(处于和串口服务器同一网段即可通信, 但不允许为网络中其他设备的IP, 避免冲突) |
| 子网掩码     | 任意值(不重要)                                                  |
| 默认网关     | 任意值(不重要)                                                  |
| 首选DNS服务器 | 任意值(不重要)                                                  |
| 备用DNS服务器 | 不填(不重要)                                                   |

3. **访问串口服务器配置界面**: 浏览器访问 `192.168.14.200`
4. **登录系统**: 用户名 `root`, 密码 `000997`
5. **配置网络接口**: 选择网络->接口, 选择LAN->修改

| 配置项      | 设置值             |
|----------|-----------------|
| 协议       | 静态地址            |
| IPv4地址   | 172.18.<工位号>.15 |
| 子网掩码     | 255.255.255.0   |
| 默认网关     | 172.18.<工位号>.1  |
| 首选DNS服务器 | 172.18.<工位号>.1  |

6. **点击保存&应用**保存更改
7. **注**: 修改完串口服务器的地址后, 应将电脑的IP地址还原为题目中规定的IP地址

## 六. 虚拟机配置

### VirtualBox虚拟机配置

1. **打开Oracle VM VirtualBox**
2. **导入虚拟机**:

    * 文件: <自行选择虚拟机系统>

|------|----------------------------------|
| IP地址 | 172.18.<工位号>.17                  |
| 子网掩码 | 255.255.255.0(必须和路由器LAN口子网掩码一致)  |
| 网关   | 172.18.<工位号>.1(必须和路由器LAN口IP地址一致) |

## 七. 摄像头配置

### TPLink摄像头配置

1. **重置TPLink摄像头**: 抬高摄像头头部, 按住露出的重置按钮不放, 听到"充值成功, 摄像机重启"后代表重置成功
2. **打开摄像头工具**: 公共工具包_全栈版\工具&驱动\01_摄像头工具\TP-LINK, 解压并打开TP-LINK安防系统 3.6.7.495.zip
3. **安装TP-LINK安防系统**
4. **打开TP-LINK安防系统**, 点击上方的设置按钮
5. **找到摄像头设备**: 设备列表\未添加设备, 找到摄像头设备(摄像头默认IP为192.168.1.60,
   有时候即使重置了也会自动更改为172.18.19.101)
6. **修改网络配置**: 点击摄像头的更多\修改网络配置选项

| 配置项 | 设置值             |
|-----|-----------------|
| IP  | 172.18.<工位号>.13 |
| 网关  | 172.18.<工位号>.1  |

7. **配置完毕**

## 九. 配置注意事项

1. **子网掩码一致性**: 所有设备的子网掩码必须保持一致(255.255.255.0)
2. **网关一致性**: 所有设备的默认网关必须指向路由器LAN口IP(172.18.<工位号>.1)
3. **IP地址规划**: 按照规划分配IP地址, 避免地址冲突, 否则可能会导致网络风暴
