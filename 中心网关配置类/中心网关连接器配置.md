# 中心(边缘)网关连接器配置

## 一. 访问网关

1. **访问网关地址**:
    * 默认地址: `192.168.1.100`
    * 比赛配置修改后: `172.18.<工位号>.16`

2. **登录网关**:
    * 用户名: `newland`
    * 密码: `newland`

## 二. 新增连接器配置

### 串口设备连接器配置

1. **点击配置**, 新增连接器(串口设备)

#### 基本配置

| 配置项     | 说明           |
|---------|--------------|
| 连接器名称   | <按题目要求进行填写>  |
| 连接器设备类型 | <根据设备协议进行填写> |
| 设备接入方式  | 串口接入/串口服务器接入 |

#### 连接器设备类型选项

| 设备类型                  | 适用设备                                |
|-----------------------|-------------------------------------|
| Modbus over Serial    | 适用于通用网关协议(GPS, 485二氧化碳可通过这种连接器进行接入) |
| Zigbee over Serial    | 适用于连接Zigbee连接器相关                    |
| UHF RFID reader       | 适用于连接UHF射频读写器                       |
| NLE MODBUS-RTU SERVER | 适用于连接串口终端                           |
| NLE SERIAL-BUS NL-UWB | 适用于连接UWB定位套件等                       |
| LRDSCREEN FOURLINE    | 适用于连接LED综合显示屏                       |

### 串口接入配置

选择"串口接入"时的配置:

| 配置项  | 说明                          |
|------|-----------------------------|
| 波特率  | <根据设备填写>                    |
| 串口名称 | 新大陆的网关目前只有一个串口, 所以选择一个固定的即可 |

#### 常见设备默认波特率

| 设备类型          | 默认波特率 |
|---------------|-------|
| 二氧化碳变送器(485型) | 9600  |
| 水浸传感器         | 9600  |
| 超声波传感器(485型)  | 9600  |
| 北斗定位模块        | 9600  |
| 多合一传感器        | 9600  |
| 光照噪声变送器       | 9600  |
| ZigBee协调器     | 9600  |

#### 波特率读取与设置方法

如果设备没有默认波特率, 可以进行以下操作:

1. **硬件连接**: 将设备通过RS232转RS485转换器接入电脑
2. **打开配置工具**: 公共工具包_全栈版\工具&驱动\33_全栈工具\水浸百叶箱传感器配置工具\485参数配置工具V3.3
3. **配置参数**:
    * 选择相应的串口号
    * 点击测试波特率
    * 成功后可对设备地址及波特率进行修改

### 串口服务器接入配置

选择"串口服务器接入"时的配置:

| 配置项     | 设置值                                        |
|---------|--------------------------------------------|
| 串口服务器IP | 172.18.<工位号>.15                            |
| 串口服务器端口 | 看串口服务器上的接口, 从RS232的4个到RS485的2个分别是6001-6006 |

## 三. 网络设备连接器配置

### 基本配置

| 配置项       | 说明                      |
|-----------|-------------------------|
| 网络设备连接器名称 | <自定义, 若题目有要求则按题目要求进行填写> |
| 网络设备连接器类型 | <根据设备进行填写>              |
| Modbus类型  | <根据设备进行填写>              |

### 网络设备连接器类型选项

| 设备类型                  | 适用设备                                                 |
|-----------------------|------------------------------------------------------|
| Modbus over TCP       | 适用于以太网(TCP/IP)连接的Modbus设备(就是有网口且可以用网口连接的设备), 例IoT采集器 |
| NLE MODBUS-RTU SERVER | 适用于其他系统主动读取网关数据, 例串口终端                               |
| DAHUA ALARM           | 仅适用于大华报警控制器私有协议, 例大华报警控制器                            |

### Modbus类型选项

| Modbus类型          | 说明                                           |
|-------------------|----------------------------------------------|
| Modbus over TCP   | 适用于以太网(TCP/IP)连接的Modbus设备(就是有网口且可以用网口连接的设备)  |
| NLE MODBUS COMMON | 新大陆标准Modbus TCP驱动模块(一般都选择这个, 因为目前只用到了新大陆的设备) |
