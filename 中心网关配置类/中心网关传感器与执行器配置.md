# 中心网关传感器与执行器配置

## 一. 确认设备连接

确保已经正确连接所有设备, 并参考[中心(边缘)网关连接器配置.txt]文件进行配置.

## 二. 串口执行器设备配置

### 串口执行器配置步骤

| 配置项  | 说明/值                                                              |
|------|------------------------ -------------------------------------------|
| 设备名称 | <根据项目要求填写>                                                        |
| 标识名称 | <根据项目要求填写>                                                        |
| 设备类型 | modbus rtu 执行器                                                    |
| 从机地址 | 十六进制设备地址(十六进制), 要求两位数字, 如果十六进制只有一位的话, 要在前面补一个0, 比如十六进制10, 十六进制为0a |
| 功能码  | 01: 线圈寄存器, 只有True/False值<br>03: 保持寄存器, 数值类型, 可以存放数值               |
| 起始地址 | 从0000开始, 第一个为0001, 第二个为0002                                       |
| 读取方式 | 目前已知一种value, 即原始值, 按设备要求写                                         |
| 写入方式 | 数据写入方式, 根据要求写, 没有就空                                               |

## 三. 网络执行器设备配置

### 网络执行器配置步骤

| 配置项  | 说明/值                                                                                           |
|------|------------------------------------------------------------------------------------------------|
| 设备名称 | <根据项目要求填写>                                                                                     |
| 标识名称 | <根据项目要求填写>                                                                                     |
| 设备IP | <172.18.<组号>.18/19>(IOT数据采集器为组内接入, 因为IOT数据采集器是中心网关与设备之间进行数据转换的, 所以设备要连接在对应的IOT上, 才能进行数据采集和控制的) |
| 设备端口 | 502(固定)                                                                                        |
| 从机地址 | 01(固定)                                                                                         |
| 功能码  | 01<br>01: 线圈寄存器, 数据类型: 只有True/False值<br>03: 保持寄存器, 数据类型: 数值类型, 按设备                             |
| 起始地址 | 继电器(直连到底层)是第几个口就是000几                                                                          |
| 读取方式 | 空                                                                                              |
| 写入方式 | 数据写入方式, 根据要求写, 没有就空                                                                            |

## 四. 传感器采集设备配置

### 传感器采集配置步骤

| 配置项  | 说明/值                                                                                                           |
|------|----------------------------------------------------------------------------------------------------------------|
| 设备名称 | <根据项目要求填写>                                                                                                     |
| 标识名称 | <根据项目要求填写>                                                                                                     |
| 设备IP | <按实际设备填写, 如果是IOT的话就填IOT的地址>                                                                                    |
| 设备端口 | <按实际设备填写, 如果是IOT的话就填IOT的端口, 默认为502>                                                                            |
| 从机地址 | IOT设备: 01<br>410s温湿度传感器: 01<br>404D设备(光照传感器): 02<br>风扇叶片: 查询具体地址, 按实际地址<br>RGB控制盒: 07<br>水浸传感器: 01<br>直流电机: 01 |
| 功能码  | <01为读取状态/执行器, 02是读取状态的寄存器, 03是读取数值的寄存器, 具体看IOT物联网数据采集设备配置文档(传感器与执行器部分)>                                        |
| 起始地址 | 继电器执行器起始地址: 0001-0008(DO 1-8)<br>数字量输入起始地址: 03e9-03f0(DI 1-8, 十进制值为1001-1008)<br>模拟量输入起始地址: 0dad-0daf(AIN 1-3) |
| 数据长度 | 0001                                                                                                           |
| 读取方式 | 目前已知一种value, 即原始值, 按设备要求写                                                                                      |
| 写入方式 | 数据写入方式, 根据要求写, 没有就空                                                                                            |

### 从机地址对应表

| 设备类型          | 从机地址          |
|---------------|---------------|
| IOT设备         | 01            |
| 410s温湿度传感器    | 01            |
| 404D设备(光照传感器) | 02            |
| 风扇叶片          | 查询具体地址, 按实际地址 |
| RGB控制盒        | 07            |
| 水浸传感器         | 01            |
| 直流电机          | 01            |

### 起始地址对应表

| 接口类型       | 地址范围      | 说明                     |
|------------|-----------|------------------------|
| 继电器执行器(DO) | 0001-0008 | DO 1-8                 |
| 数字量输入(DI)  | 03e9-03f0 | DI 1-8, 十进制值为1001-1008 |
| 模拟量输入(AIN) | 0dad-0daf | AIN 1-3                |
