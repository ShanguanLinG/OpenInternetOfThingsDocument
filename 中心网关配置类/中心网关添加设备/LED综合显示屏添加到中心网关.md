# LED综合显示屏添加到中心网关

## 一. 准备工作

1. **将LED综合显示屏使用USB转接器连接到电脑上**
2. **打开工具路径**: 公共工具包_全栈版\工具&驱动\33_全栈工具\综合显示屏\T1D系列控制卡二次开发包V2.5A\LEDCenterV1.22

## 二. 显示屏配置

### 设置屏幕参数

1. **进入设置界面**: 点击设置->输入密码888
2. **更改屏幕尺寸**:
    * 长度: 128
    * 高度: 64
3. **保存配置**: 依次点击发送后保存, 保存配置

## 三. 硬件连接

* **连接显示屏到中心网关**: 将综合显示屏的USB接口接到中心网关的USB接口上

## 四. 中心网关配置

### 访问配置界面

* **打开物联网中心网关配置界面**: 172.18.<工位号>.16

### 新增连接器配置

选择配置->新增连接器, 选择串口设备:

| 配置项     | 设置值                 |
|---------|---------------------|
| 连接器名称   | 综合显示屏               |
| 连接器设备类型 | LEDSCREEN FOUR LINE |
| 设备接入方式  | 串口接入                |
| 波特率     | 9600                |
| 串口名称    | 选择名称中带USB的          |

### 新增LED设备

找到连接器, 点击新增LED:

| 配置项  | 设置值          |
|------|--------------|
| 传感名称 | 综合显示屏        |
| 表示名称 | <按照题目要求进行填写> |
| 序列号  | 01           |
| 传感类型 | 四行LED        |
