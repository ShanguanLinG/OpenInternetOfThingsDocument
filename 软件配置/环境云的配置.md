# 环境云的配置

## 一. 软件安装与启动

1. **安装环境云软件**并进入系统

## 二. 创建场景

1. **点击左上角的按钮**添加场景
2. **填写场景名称**: 按题目要求进行填写
3. **点击保存**

## 三. 硬件连接配置

### NSLora设备连接

1. **连接NSLora至电脑**:
    * **找到NSLora设备接口**: 设备上的标识(蓝色)485A, 485B
    * **连接线缆**: 使用这两个接口去接一根红黑线连接到RS485转232接口上
    * **电源连接**: 当NSLora板的电源接完之后
    * **串口转换**: 将485A和485B连接RS485转232接口连接USB转串口
    * **连接电脑**: 将USB转串口的USB一端连接到电脑上

### 软件串口配置

1. **打开智能环境云**, 点击右上角的配置
2. **配置串口参数**:

| 配置项  | 说明                                                                          |
|------|-----------------------------------------------------------------------------|
| 串口   | 右键此电脑->管理->设备管理器->端口选项(COM和LPT), 找到设备对应的COM<数字>, 就是串口号(不知道是哪个的可以重新插拔一下USB线) |
| 服务端口 | 默认即可                                                                        |

## 四. 添加设备配置

### 设备基本信息配置

1. **点击智能环境**, 进行添加设备:

| 配置项  | 说明                                         |
|------|--------------------------------------------|
| 标识码  | <按题目要求进行填写>                                |
| 设备名称 | <按题目要求进行填写>                                |
| 地址   | <NS Lora设备的地址(NSLora显示屏上的Addr(地址)值, 一般为1)> |
| 通道   | 第一个设备就是0, 第二个设备是1, 以此类推, 最多为4              |
| 数据范围 | <按题目要求进行填写>                                |

### 随机数据配置

| 配置项        | 说明          |
|------------|-------------|
| 最小值        | <按题目要求进行填写> |
| 最大值        | <按题目要求进行填写> |
| 小数位        | <按题目要求进行填写> |
| 每次生成的值不相差± | <按题目要求进行填写> |

### 通信配置

| 配置项  | 说明          |
|------|-------------|
| 连接方式 | 串口          |
| 发送模式 | <按题目要求进行填写> |
| 发送间隔 | <按题目要求进行填写> |

2. **点击保存**

## 五. 启动与运行

### 启动顺序

1. **重启环境云**
2. **重启NS-Lora设备**
3. **点击"运行"**发送数据到设备中
